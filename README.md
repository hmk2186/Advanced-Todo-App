# 📝 Advanced To-Do List App

A clean, responsive and user-friendly To-Do List application built using **React** and **Tailwind CSS**.
This app helps users manage tasks efficiently with features like search, filtering, priorities, and deadlines.

---

## 🌐 Live Demo

🚀 **Try the App Here:**
👉 https://advanced-todo-app-five.vercel.app/

---

## 🚀 Features

* ➕ Add new tasks
* ✏️ Edit existing tasks
* 🗑️ Delete tasks
* ✅ Mark tasks as completed
* 📅 Set deadline (date & time)
* 🔍 Search tasks by text
* 🎯 Filter tasks by status:

  * All
  * Completed
  * Active
* ⚡ Set priority:

  * Low
  * Medium
  * High
* 🏷️ Categorize tasks:

  * Work
  * Personal
  * Study
* ⏳ Automatic status detection:

  * Pending
  * Completed
  * Overdue
* 📱 Fully Responsive UI:

  * Mobile → Card layout
  * Tablet/Desktop → Table layout

---

## 🛠️ Tech Stack

* ⚛️ React (Functional Components + Hooks)
* 🎨 Tailwind CSS
* 🔢 UUID (for unique task IDs)

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/your-username/advanced-todo-app.git

# Navigate to project folder
cd advanced-todo-app

# Install dependencies
npm install

# Run the app
npm run dev
```

## ⚠️ Current Limitations

* ❌ No backend (data not saved permanently)
* ❌ No authentication/login system
* ❌ No localStorage for tasks (refresh clears data)
